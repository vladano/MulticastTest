Test Example

To test multicast IP network traffic on IP address 224.0.0.1 and port 5000, we need to use the following syntax:

For receiving data:

MulticastReceive.exe -a 224.0.0.1 -p 500

MulticastSend.exe -a 224.0.0.1 -p 5000 –t 3 –n 5 –s 10
